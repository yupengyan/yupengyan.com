220.243.135.0 这类规律性的IP可能就是爬虫了…可以用Haprox等web应用服务器block,也可以用防火墙block，当然也有可能是搜索引擎，block前搞清楚。

处理¶

  1. 放入防火墙拒绝 
firewall-cmd --permanent --add-rich-rule="rule family='ipv4' source address='180.76.15/24' reject"
firewall-cmd --reload
  1. 放入haproxy黑名单
    haproxy.cfg frontend 放入： 
    # Black list ######
    acl bad_ip src -f /usr/local/etc/haproxy/block-ip.txt
    tcp-request connection reject if bad_ip

block-ip.txt 放入ip地址： 

220.243.135.0/24
